Welcome to my blog! This is my first post.

## What to expect

I'll be writing about things I'm working on, learning, or just find interesting. No strict schedule — just whenever I have something worth sharing.

## How this works

This blog is built with [Jekyll](https://jekyllrb.com/) and hosted for free on [GitHub Pages](https://pages.github.com/).

Writing a new post is as simple as creating a new `.md` file in the `_posts/` folder with this format:

```
_posts/YYYY-MM-DD-title-of-post.md
```

And starting it with this header (called "frontmatter"):

```yaml
---
layout: post
title: "Your Post Title"
date: 2026-07-20
categories: general
---
```

That's it. Push to GitHub and it's live.
